Animations

Animation patterns and keyframes used across the site.

Fade In

Used for page load animations and scroll reveals.

  from { opacity: 0; transform: translateY(20px); }
  to { opacity: 1; transform: translateY(0); }
}

Classes: .animate-fade-in, .animate-fade-in-delay-1 through .animate-fade-in-delay-4

Used for status indicators and attention-grabbing elements.
